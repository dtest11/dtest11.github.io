超时/重传

1.计时器

tcp在发送数据的时候会启动一个计数器,若到时间还没有收到回复,那么会触发相应的超时或者重传

2.重传二义性

假如一个包出现超时,那么就会触发重传,过一段时间会收到一个包的确认,那么此时的这个包是对于第一次发送的确认,还是对与重传包的确认,这个确认就会存在二义性.
